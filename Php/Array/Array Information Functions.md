
`count()` - Counts the number of elements in an array. 
```php
$numbers = array(1, 2, 3, 4, 5); 
echo count($numbers)
```

`sizeof()` - An alias of count() . Also counts the number of elements in an array. 
```php
$numbers = array(1, 2, 3, 4, 5); 
echo sizeof($numbers);
```
